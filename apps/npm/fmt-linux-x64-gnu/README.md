# @rsvelte/fmt-linux-x64-gnu

Prebuilt [`@rsvelte/fmt`](https://www.npmjs.com/package/@rsvelte/fmt) binary for **Linux x64** (glibc).

**Do not install this package directly.** Install the loader package:

```bash
npm install -D @rsvelte/fmt
```

The loader will pull in the correct platform binary (this one, if you're on glibc Linux x64) via `optionalDependencies` and invoke it transparently.

Part of the [rsvelte](https://github.com/baseballyama/rsvelte) project — a Rust port of the Svelte 5 compiler and surrounding toolchain.
